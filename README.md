1. 什么时候止盈：上涨过程中，只能在第二个“突然下跌”时卖出。
2. 止损：买入前便要设置好收盘价跌到多少便立马平仓。 
3. 不可补仓：只能追涨杀跌，越涨越买。
4. 保持身体健康：生病的时候不准交易。
5. 不要在意单次交易亏损，而是关注自己单次交易是否严格遵循规则。
6. 不要苛求局部最优，只求全局最优。心态上完全接受单次交易只是一个随机的概率，亏损了也没关系。
7. 承担责任。不管结果是什么，都是自己当前水平的完美反映，是为了让自己学习以便做得好。
8. 在入场那一刻，就在心理上（和账户上）完全放弃这笔止损额度。心理上认为这比亏损就是交易成本，帐户上必须在入场后提交止损单。
9. 当天必须进行一次“无头寸”复盘，也就是假设自己没有持有这些股票，我是否会再次买入这些股票。
10. 当对对感知到的信息感到恐惧或过度兴奋时，问问自己：这个信息本身有威胁吗，还是你只是体验到反射回来的你自己的思想状态（近期亏损的恐惧状态、近期盈利的过度兴奋状态）。见附录1
