1. 什么时候止盈：上涨过程中，只能在第二个“突然下跌”时卖出。
2. 止损：买入前便要设置好收盘价跌到多少便立马平仓，而且除了在大幅盈利的价位之外，只有在止损位才能平仓。
3. 不可补仓：只能追涨杀跌，越涨越买。
4. 必须提前计算风险（盈亏概率）。
5. 必须计划交易，也就是在前一天晚上确定，在某一股票xx价位买卖/在某一股票xx时间买卖。
6. 保持身体健康：
   1. 生病的时候不准交易。
   2. 设立目标，每天坚持跑步。
7. 不要在意单次交易亏损，而是关注自己单次交易是否严格遵循规则。
8. 微观随机，宏观一致：微观上，每一次交易的结果都是不确定的，因而不必在意；宏观上，只要保证了优势概率，最终都能走向盈利。所以根本不必在意每一次的输赢。
9. 定义成功：
   1. 成功不是下一笔交易必须盈利，而是实现一致性盈利，而遵守交易规则是唯一的方式。
   2. 不要把自尊心绑定在最近几次的交易上，应该把自尊心绑定在自己是否完全按规则执行交易上。
10. 赢家态度：承担责任。不管结果是什么，都是自己当前水平的完美反映，是为了让自己学习以便做得好。
11. 止损位是生意成本：在入场那一刻，就在心理上（和账户上）完全放弃这笔止损额度。心理上认为这比亏损就是交易成本，帐户上必须在入场后提交止损单。
12. 客观观察自己的思想、言词、行动：
   1. 当天必须进行一次“无头寸”复盘，也就是假设自己没有持有这些股票，我是否会再次买入这些股票。
   2. 利用github博客客观记录观察你的思想。
12. 保持心态平静：
    1. 当对对感知到的信息感到恐惧或过度兴奋时，问问自己：这个信息本身有威胁吗，还是你只是体验到反射回来的你自己的思想状态（近期亏损的恐惧状态、近期盈利的过度兴奋状态）。见附录1。
    2. “避免自毁信念”：当有自毁信念出现，马上关闭电脑，去江边跑步散步。常见自毁信念：“反正都亏这么多了，再亏多点也一样。”
14. 五大基本事实
    1. 任何事都有可能会发生。
    2. 为了挣钱，你不必知道下一步会发生什么。（不要期望市场发生什么、不要觉得你知道市场会怎么走）
    3. 任何一组决定胜算的变量所带来的输赢在分布上都是随机的。（微观上交易完全随机）
    4. 任何胜算都仅仅是表明发生某种事件的概率比较高。
    5. 市场在任何一个时刻都是独一无二的。
      1. 不能把市场拟人化，市场只是无限的机会流，不会表达任何东西。
